# Plataforma Educacional Interativa – Instituto Semente da Vida

Este projeto consiste no desenvolvimento de uma plataforma web educacional interativa voltada à conscientização de crianças e adolescentes em situação de vulnerabilidade sobre limites, proteção e identificação de situações inadequadas.

A solução foi desenvolvida como parte de um projeto de extensão do curso de Análise e Desenvolvimento de Sistemas (ADS), em parceria com o Instituto Semente da Vida, localizado em Várzea Paulista.

## 🚀 Objetivo

Transformar materiais educativos em uma experiência digital acessível, organizada e intuitiva, permitindo que educadores e jovens naveguem por conteúdos de forma estruturada e segura.

## 🧩 Funcionalidades

- Navegação por faixas etárias:
  - 11–14 anos
  - 15–17 anos
- Organização por módulos educativos
- Sistema de exibição de slides (imagens)
- Navegação entre slides (próximo/voltar)
- Indicador de progresso (slide atual)
- Interface responsiva (desktop e mobile)

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 🎯 Diferenciais

- Foco em impacto social real
- Interface simples, acessível e intuitiva
- Estrutura escalável para futuras melhorias (como gamificação e interações)
- Otimização para leitura e compreensão do conteúdo educativo

## 🌐 Deploy

O projeto pode ser hospedado em plataformas como Netlify ou Vercel para acesso online.

## 📌 Contexto

Este projeto foi desenvolvido com base em um diagnóstico real realizado junto à ONG, que identificou a necessidade de educação preventiva para crianças que muitas vezes não reconhecem situações de risco como inadequadas.

## 🤝 Impacto

A plataforma contribui para:

- Educação preventiva
- Fortalecimento da autonomia
- Promoção de segurança e consciência social
